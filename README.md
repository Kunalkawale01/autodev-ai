# 🤖 AutoDev AI

**AutoDev AI** is an autonomous AI developer that can analyze tasks, simulate development workflows, and generate structured outputs like a real engineering team.

It acts like a **virtual dev team** — Builder, Reviewer, and Tester — all in one system.

---

## Features

**Bug Fixing** – Detects and resolves logical issues
**Feature Generation** – Simulates building new features
**Performance Optimization** – Improves efficiency
**Code Review** – Suggests improvements and best practices
**Test Generation** – Creates validation test cases
**Memory System** – Stores past tasks and activity
**Voice Input** – Speak tasks directly (browser-supported)
**Premium UI** – Clean, modern dashboard interface

---

## Tech Stack

* **Backend:** Node.js + Express
* **Frontend:** HTML, CSS, Vanilla JavaScript
* **Storage:** Local JSON (memory system)

---

## How It Works

AutoDev AI processes your input and routes it through simulated agents:

* **Builder Agent** → Implements changes
* **Reviewer Agent** → Improves code quality
* **Tester Agent** → Validates functionality

It then stores the task in memory for tracking and analytics.
